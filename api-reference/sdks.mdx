## Node.js

Install from npm:

```bash
npm install firstquadrant
```

Import and authenticate:

```ts
import { FirstQuadrant } from "firstquadrant";

const client = new FirstQuadrant("<API key>");
await client.getTeam(); // Returns the current team
```
